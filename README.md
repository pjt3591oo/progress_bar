# progress bar

progress bar for loading

- install

```bash
$ pip install progress_bar
```

- usage

```python
from progress_bar import progress_bar as pb

pb.progress_bar(1)
pb.progress_bar(2)
pb.progress_bar(3)
pb.progress_bar(4)
pb.progress_bar(5)
pb.progress_bar(6)
pb.progress_bar(7)
```
